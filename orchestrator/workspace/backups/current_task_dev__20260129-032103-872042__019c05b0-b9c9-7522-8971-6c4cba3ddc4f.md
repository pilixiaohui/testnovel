## 实现要求

### 1. 创建 components/editor 目录结构
将编辑器组件放入 `src/components/editor/` 目录。

### 2. 创建 SceneEditor.vue
场景编辑器组件，包含：
- 场景内容编辑区域（textarea 或富文本）
- 保存/取消按钮
- Props: `sceneId: string`, `content?: string`
- Emits: `save`, `cancel`

### 3. 创建 ContentRenderer.vue
内容渲染器组件，包含：
- 渲染场景正文内容
- 支持 Markdown 或纯文本
- Props: `content: string`, `format?: 'markdown' | 'text'`

### 4. 创建 VersionDiff.vue
版本对比组件，包含：
- 左右对比布局
- 高亮差异部分
- Props: `oldContent: string`, `newContent: string`

### 5. 创建 MarkdownPreview.vue
Markdown 预览组件，包含：
- 实时预览 Markdown 内容
- Props: `content: string`

### 6. 创建 SimulationLogs.vue
推演日志查看组件，包含：
- 展示推演回合列表
- 每回合显示 agent_actions 和 dm_arbitration 摘要
- Props: `logs: SimulationRoundResult[]`

### 7. 创建 SceneContextPanel.vue
场景上下文面板组件，包含：
- 展示场景上下文信息（entities, relations, prev/next scene）
- Props: `context: SceneContextView`
