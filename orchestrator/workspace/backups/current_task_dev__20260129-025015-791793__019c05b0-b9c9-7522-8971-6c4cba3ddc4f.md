### 1. 创建 components/snowflake 目录结构

### 2. 创建 StepIndicator.vue
步骤指示器组件，包含：
- 显示当前步骤（1-6）
- 步骤标题和状态（完成/进行中/待完成）
- 使用 Element Plus 的 el-steps 组件
- Props: `currentStep: number`, `steps: Array<{title: string, description?: string}>`

### 3. 创建 Step1Panel.vue - Step6Panel.vue
6 个步骤面板组件：
- Step1Panel: 想法输入面板（el-input textarea + 生成按钮）
- Step2Panel: 结构展示面板（展示 logline/theme/ending/three_disasters）
- Step3Panel: 角色列表面板（展示 CharacterCard 列表）
- Step4Panel: 场景列表面板（展示 SceneCard 列表）
- Step5Panel: 幕章结构面板（展示 ActCard/ChapterCard）
- Step6Panel: 锚点生成面板（展示锚点列表）

### 4. 创建 LoglineSelector.vue
Logline 选择器组件，包含：
- 展示多个 logline 选项（el-radio-group）
- 选中回调
- Props: `options: string[]`, `modelValue: string`
- Emits: `update:modelValue`

### 5. 创建 ActCard.vue
幕卡片组件，包含：
- 展示幕信息（sequence, title, purpose, tone）
- 使用 el-card
- Props: `act: Act`

### 6. 创建 ChapterCard.vue
章卡片组件，包含：
- 展示章信息（sequence, title, focus, pov_character_id）
- 使用 el-card
- Props: `chapter: Chapter`

### 7. 移动现有组件
若 `CharacterCard.vue` 和 `SceneCard.vue` 已存在于 components 根目录，移动到 `components/snowflake/` 目录。
