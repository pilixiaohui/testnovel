用户可以创建和编辑角色档案。

Scenario: Create main character
Given 已有故事大纲
When 用户创建主角"叶无痕"
Then 角色档案包含姓名、背景、动机、性格特征
And 角色与故事冲突关联

Scenario: Create supporting characters
Given 已有主角
When 用户批量创建配角（师父、反派、红颜知己）
Then 每个角色有独立档案
And 角色关系图正确建立
